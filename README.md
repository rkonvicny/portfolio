# Osobní portfolio

Moderní, plně responzivní a data-driven webová prezentace pro softwarového vývojáře a architekta. Tento projekt slouží jako digitální vizitka a rychlý přehled zkušeností.

🌐 **Živá ukázka:** [Zde bude doplněn odkaz po nasazení]

## 🛠️ Použitý Tech Stack

- **Next.js (App Router)** – Výkonný React framework.
- **TypeScript** – Typová bezpečnost a robustnost.
- **Tailwind CSS v4** – Moderní a optimalizovaný design bez nutnosti psát dedikované CSS.
- **Clean Architecture** – Kód je striktně rozvrstven (Domain, Application, Infrastructure), takže je UI odděleno od business logiky a napojení třetích stran (SMTP, Databáze).

## 📄 Funkcionalita

- Generování obsahu plně z centrálního `portfolio.json` souboru.
- Plně funkční kontaktní formulář (na pozadí odesílá zprávy přes SMTP).
- Elegantní, čisté CSS animace a Dark/Light motiv.
